## 🚀 Featured Projects

### 🔍 RAG Pipeline — Smart Document Q&A
> A production-ready Retrieval Augmented Generation pipeline that lets you chat with your documents.

- Built with **LangChain + FAISS + OpenAI / local LLMs**
- Supports PDF, DOCX, and web URL ingestion
- Chunking strategies, re-ranking, and hybrid search
- **Python CLI interface** for ease of use

`Python` `LangChain` `FAISS` `RAG` `CLI`

---

### 🤖 Agentic AI Assistant — Multi-Tool Agent
> An autonomous agent that plans, reasons, and uses tools to complete complex tasks.

- Implements **ReAct + Tool-use** architecture
- Integrates with web search, SQL databases, and file systems
- Built using **MCP (Model Context Protocol)** for tool orchestration
- Stateful memory across sessions

`Python` `MCP` `Agentic AI` `LangChain` `FastAPI`

---

### 🧠 LLM Fine-tuning Pipeline
> End-to-end pipeline to fine-tune open-source LLMs on custom datasets.

- Supports **LoRA / QLoRA** for efficient fine-tuning
- Dataset preparation, training, evaluation, and GGUF export
- Works with **Mistral, LLaMA, Phi** model families
- CLI tool for training config management

`Python` `HuggingFace` `LoRA` `QLoRA` `LLM`

---

### 💻 AI-Powered CLI Toolkit
> A suite of command-line tools powered by LLMs for developer productivity.

- Code review, docstring generation, commit message writer
- Built with **Typer + Rich** for beautiful terminal UIs
- Supports multiple LLM backends (OpenAI, Ollama, Anthropic)
- Config management with SQL-backed history

`Python` `Typer` `Rich` `CLI` `SQL`
